# Drum Mappings

## Input to Song Data Normalization

Maps all input device drum names to standardized song data format.

| Input Name | Normalized Name |
|---|---|
| Rack Tom 1 | rack tom 1 |
| Rack Tom 2 | rack tom 2 |
| Floor Tom 1 | floor tom 1 |
| Floor Tom 2 | floor tom 2 |
| Snare Drum | snare |
| Ride | ride |
| Crash Cymbal 1 | crash cymbal 1 |
| Crash Cymbal 2 | crash cymbal 2 |
| Hi-Hat_1 | hi-hat 1 |
| Hi-Hat_2 | hi-hat 2 |
| Bass Drum | bass |
